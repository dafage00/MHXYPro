# 小说阅读器

一个功能完整的桌面小说阅读程序，支持自动获取和阅读小说。

## 功能特点

- 📚 **本地小说阅读** - 支持读取本地TXT格式小说文件
- 🔍 **智能章节识别** - 自动识别小说章节结构
- 📖 **流畅阅读体验** - 简洁美观的阅读界面
- 🎤 **语音朗读功能** - 支持男声/女声朗读，可调节语速
- ✨ **实时高亮** - 朗读进行到哪一段就自动高亮文本位置
- 💾 **阅读进度保存** - 自动保存阅读进度，下次继续阅读
- 📝 **阅读历史记录** - 记录最近阅读的章节
- 🗂️ **小说管理** - 方便管理多本小说
- 🔖 **书签功能** - 添加、管理和跳转书签
- 🔎 **全文搜索** - 在当前章节中搜索关键词
- 📊 **阅读统计** - 记录阅读时长、字数统计和阅读速度
- ⌨️ **快捷键支持** - 支持常用快捷键操作
- 🔤 **字体调节** - 可调节字体大小，适应不同阅读习惯

## 安装说明

### 1. 安装Python

确保您的电脑已安装Python 3.7或更高版本。

### 2. 安装依赖

在项目目录下运行：

```bash
pip install -r requirements.txt
```

## 使用方法

### 启动程序

运行主程序：

```bash
python novel_reader.py
```

### 添加小说

1. 点击"添加小说"按钮
2. 选择本地TXT格式的小说文件
3. 程序会自动识别小说标题和章节结构

### 阅读小说

1. 在左侧列表中选择要阅读的小说
2. 双击小说名称或选择后点击打开
3. 使用"上一章"和"下一章"按钮切换章节
4. 也可以使用章节下拉框快速跳转到指定章节

### 删除小说

1. 在左侧列表中选择要删除的小说
2. 点击"删除小说"按钮
3. 确认删除操作

### 语音朗读

1. 打开要朗读的小说章节
2. 在"语音朗读"区域选择声音类型（男声/女声）
3. 使用语速滑块调节朗读速度（50-300）
4. 点击"开始朗读"按钮开始朗读当前章节内容
5. 点击"停止"按钮可随时停止朗读
6. 朗读时界面会自动高亮当前朗读位置，方便跟读

**注意**：语音朗读功能需要安装 `pyttsx3` 库。如果未安装，相关功能将被禁用。

### 书签功能

1. 在阅读时，点击"添加书签"按钮或按 `Ctrl+B` 添加当前章节的书签
2. 点击工具栏的"书签"按钮查看所有书签
3. 在书签列表中双击书签或点击"跳转"按钮可快速跳转到该章节
4. 可以删除不需要的书签

### 搜索功能

1. 在搜索框中输入关键词
2. 点击"搜索"按钮或按 `Enter` 键开始搜索
3. 使用"上一个"和"下一个"按钮或按 `F3`/`Shift+F3` 切换搜索结果
4. 搜索结果会高亮显示，并自动滚动到匹配位置

### 阅读统计

1. 点击工具栏的"统计"按钮查看阅读统计
2. 统计信息包括：
   - 总阅读时长
   - 总阅读字数
   - 平均阅读速度
3. 阅读统计会自动记录，无需手动操作

### 字体调节

1. 使用"A+"和"A-"按钮调节字体大小
2. 也可以使用快捷键 `Ctrl++` 和 `Ctrl+-` 调节
3. 字体大小范围：8-30

### 快捷键

- `Ctrl+F` - 聚焦搜索框
- `Ctrl+Left` - 上一章
- `Ctrl+Right` - 下一章
- `Ctrl++` - 增大字体
- `Ctrl+-` - 减小字体
- `Ctrl+B` - 添加书签
- `F3` - 搜索下一个
- `Shift+F3` - 搜索上一个

## 支持的章节格式

程序支持以下章节标题格式：

- `第一章 标题`
- `第1章 标题`
- `Chapter 1 标题`
- `第 1 章 标题`

如果小说没有章节标记，程序会自动按段落分割成多个部分。

## 文件结构

```
.
├── novel_reader.py      # 主程序（GUI界面）
├── novel_fetcher.py     # 小说获取模块
├── novel_manager.py     # 小说管理模块
├── requirements.txt     # 依赖包列表
├── README.md           # 使用说明
└── novels_data/        # 数据存储目录（自动创建）
    └── novels_db.json  # 小说数据库
```

## 技术栈

- **Python 3.7+**
- **tkinter** - GUI界面
- **requests** - HTTP请求
- **BeautifulSoup4** - HTML解析

## 注意事项

1. 程序会自动在运行目录下创建`novels_data`文件夹存储数据
2. 阅读进度会自动保存，下次打开同一本小说时会自动定位到上次阅读的位置
3. 支持UTF-8编码的TXT文件，如果文件编码不同可能需要转换
